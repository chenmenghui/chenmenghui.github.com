```php
<?php
$a = [1, 2, 3, 4 => 7, 't' => 10,];
$b = [3, 5, 4 => 6, 't' => 11];
print_r($a + $b);
print_r((array_merge($a, $b)));
```

```result
Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [4] => 7
    [t] => 10
)
Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 7
    [t] => 11
    [4] => 3
    [5] => 5
    [6] => 6
)
```

php7.2

| 同键名 | array_merage | + |
| --- | --- | --- |
| 索引数组 | 不覆盖 | 覆盖|
| 关联数组 | 后者覆盖前者 | 前者覆盖后者 |
|索引数组返回值 | 重新计算索引 | 保留原有索引 |
