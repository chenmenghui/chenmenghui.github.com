git合并本地的几个commit操作

如果远程仓库里没有东西

  1. 进入本地指定目录
  2. git init 创建本地仓库
  3. 把本地仓库中的文件们加入本地仓库 git add .
  4. 关联远程仓库 git remote add origin git@git.oschina.net:yourname/demo.git
  5. 更新 git pull
